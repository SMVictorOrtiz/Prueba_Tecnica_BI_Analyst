# Prueba_Tecnica_BI_Analyst
Análisis del dataset de Olist (obtenido de kraggle) utilizando SQL, Python/pandas y Power BI. Incluye consultas SQL, notebook de análisis, cálculo de métricas como GMV y tiempos de entrega, y un dashboard interactivo en Power BI. El repositorio contiene instrucciones claras para ejecutar cada componente y reproducir el proyecto completo.
